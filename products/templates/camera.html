Surveillance Cameras

CCTV Cameras, Surveillance cameras monitors, records and stores events. The system comprises of a Digital Video Recorder (DVR)/ Network Video Recorder (NVR) that processes the recordings to HD( high definition) visual signal stream. Particular cameras are used for specific accusation that is the system operates day and night (24hrs), In this case infrared cameras are used, and we supply and install a wide range of security cameras e.g. Speed Dome Cameras (Pan Tilt Zoom), HD Bullet Cameras ( outdoor), IP Cameras, SPY Cameras, Wireless Cameras, HD Dome Cameras ( indoor)

The CCTV system can be operated remotely through wireless connection (Internet) .It is possible via computer and mobile phone (Smart phone). The video servers have fast genuine adaptors that link surveillance system to the IP/TPC network. The system supports local network connectivity and wide area; it has a server that transmits via TCP/IP and Modem. Remote viewing is done via, remote playback client, multicam remote view and the web browsers through the Internet or inthnet.
